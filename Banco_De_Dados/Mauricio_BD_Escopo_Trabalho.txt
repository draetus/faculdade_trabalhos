O escopo escolhido foi a administração de um servidor de um jogo, com foco nas contas dos usuários e funcionamento interno do jogo.

Foi escolhido pelo fato de que varios usuários usam o mesmo servidor, sem contar com moderadores que também o usam
é necessário um banco de dados para manter esses dados de conta ou os proprios dados que os jogos usam dentro do funcionamento,
informações de atributos, de personagens não criados por usuários ou de objetos existentes no jogo.

No banco de dados o Jogo seria a entidade inicial, tendo varios servidores, o usuario cria uma conta e os dados dessa conta são apenas para um servidor,
podendo ter varios personagens que terão habilidades, itens, talvez participem de uma guilda(grupo de jogadores), esses personagens navegam por mapas
que possuem objetos que podem ter itens, esses que tem atributos que interferem nos atributos do Personagem.
